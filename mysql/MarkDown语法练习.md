# 6.有序和无序列表
### 无序列表
```Markdown
无序列表的嵌套
* 1层
	* 1.1层
		* 1.1.1层
* 2层
* 3层
	* 3.1层
```
* 1层
	* 1.1层
		* 1.1.1层
* 2层
* 3层
	* 3.1层
### 有序列表
```Markdown
有序列表的嵌套
1. 项目一
2. 项目二
3. 项目三
```
1. 项目一
3. 项目二
5. 项目三
